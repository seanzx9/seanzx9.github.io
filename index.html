My Projects

HabTrack
Programmed in Java - A social networking Android app that allows users to track personal habits that can be shared to their network for others to see and promote. The app uses Firebase cloud services to authenticate users using official providers and to manage the Firestore database using separately created backend functions.
Multifunctional Calculator
Programmed in Java - An Android calculator app featuring 5 main functions: scientific calculator, tips calculator, price change calculator, bitwise calculator, unit converter.
Pipelined Multimedia Unit
Programmed in VHDL - A system that processes a reduced set of multimedia mips instructions (from assembler written in Java) using a 4 stage pipeline with hazard prevention.
Scrum Hub
Programmed in Java and used Firebase cloud services - A project management app that organizes project tasks in a scrum board style layout and allows team members to access projects across different Android devices.
Sonar System
Programmed in C/C++ and Java and built with a custom PCB designed in AutoDesk Eagle - A system that scans 180 degrees and processes the data to a GUI displaying the distance of objects and whether the object is inanimate or living.
Direct Digital Synthesis System
Programmed in VHDL and Synthesized to a Lattice LCMXO3L-6900C FPGA - A system that generates a selected sine, triangle, or square wave using dynamic output frequencies (user defined).
Trip Planner Interface
Programmed in Java - A program that implements a GUI in JavaFX that is used to plan trips across the USA using geographical coordinates to locate cities.
Frequency Meter System
Programmed in Assembly to an ATmega324A Microcontroller - A system with precision enhancing autoranging capability that displays on an LCD the frequency and period of an input frequency.
LED Clock
Programmed in Assembly and simulated in PSpice - a fully functional digital clock with basic timekeeping functions, alarm function, a USB charging port, and LED binary seconds counter.
